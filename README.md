### Why Echoslate?
- **Blazing fast**: <3-second startup, instant save/load â€” pure WPF + .NET 8 (no Electron)
- **Keyboard-driven**: Full hotkeys (Alt+H/L tabs, Alt+J/K severity, Ctrl+Enter complete, etc.)
- **Dev-focused features**:
  - Severity colors + per-filter ranking
  - Tags (#BUG #FEATURE #TODOS) for sorting and filtering
  - Problem/solution notes + time tracking
